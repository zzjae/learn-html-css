###### Likelion Front-end School

# HTML/CSS

멋쟁이 사자처럼 프론트엔드 스쿨 HTML/CSS 학습 저장소

## 학습 저장소 복제하기

```sh
git clone https://github.com/seulbinim/ssam-html-css.git
```

## 최신 커밋을 로컬 저장소로 가져오기

```sh
git pull origin main
```

## 의존성 패키지 설치하기

```sh
npm install
```
또는
```sh
npm i
```
**패키지를 설치하기 전 반드시 git pull 명령을 사용하여 최신 내용으로 업데이트 필요**
